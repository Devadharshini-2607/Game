#Number guessing Game
This is a simple Number Guessing Game built using Python.
The computer randomly selects a number between 1 and 100, and the player tries to guess it.

The program gives hints like:

"Too High"
"Too Low"

until the correct number is guessed.

This project is great for beginners to practice:

Python basics
Loops
Conditions
User input
Random number generation
Features

 Random number generation
 User interaction
 Unlimited guessing attempts
 Hint system (high/low)
 Simple game logic
